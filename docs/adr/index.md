# Architecture Decision Records

ADRs are append-only records of significant design decisions in `typio-settings` — the GTK4 settings panel for the Typio input method. Once accepted, they are not edited; a new ADR supersedes an old one if a decision changes.

| ADR | Title | Status |
|-----|-------|--------|
| [ADR-0001](0001-record-architecture-decisions.md) | Record Architecture Decisions | Accepted |
| [ADR-0002](0002-independent-settings-panel-repository.md) | Independent settings-panel repository and `typio-settings` naming | Accepted |
| [ADR-0003](0003-tip-v1-uds-client.md) | TIP v1 UDS client — drop GDBus, push events replace property polling | Accepted |
